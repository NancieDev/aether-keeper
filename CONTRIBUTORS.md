# Contributors

We recognize and thank all contributors who have helped make Aether Keeper better.

## How to Contribute

See [CONTRIBUTING.md](./CONTRIBUTING.md) for detailed guidelines on:
- Setting up your development environment
- Code standards and best practices
- Testing and verification requirements
- Pull request process

## Contributor Recognition

All contributions—code, documentation, bug reports, and feature requests—are valued and acknowledged in the project.

Contributors will be recognized in:
- GitHub commit history
- Release notes
- Project discussions

## Getting Started

1. Pick an issue or feature to work on
2. Read [CONTRIBUTING.md](./CONTRIBUTING.md)
3. Create a branch and make your changes
4. Submit a pull request
5. Engage in code review

**Questions?** Open an issue or check [QUICKSTART.md](./QUICKSTART.md)

Thank you for contributing! 🙏
